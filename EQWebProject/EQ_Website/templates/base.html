HOW TO GET STARTED

  1. You need to setup a EQEMU Login and a Server Login account (see step 3) on EQEMU: Register EQEMU
  2. Once you are registered, you need to go log back into EQEMU
  3. On the left navigation bar, under Accounts-Loginserver Accounts-select Create Account.
  4. You will need to dowload this Johns EQEMU zip and copy the contents to the directory you put your games
